# G-nyang's Portfolio 블로그

G-nyang의 포트폴리오 블로그입니다.

## 🌐 블로그 보기

### 온라인에서 보기 (GitHub Pages)
블로그는 다음 URL에서 확인할 수 있습니다:
- **https://g-nyang.github.io/**

GitHub에 코드를 푸시하면 자동으로 블로그가 업데이트됩니다.

### 로컬에서 실행하기

로컬 환경에서 블로그를 미리 보고 싶다면:

1. **Jekyll 설치** (Ruby 필요)
   ```bash
   gem install jekyll bundler
   ```

2. **의존성 설치**
   ```bash
   bundle install
   ```

3. **로컬 서버 실행**
   ```bash
   bundle exec jekyll serve
   ```

4. **브라우저에서 열기**
   - http://localhost:4000 에서 블로그를 확인할 수 있습니다.

## 📝 블로그 수정하기

- `index.md`: 메인 페이지 내용
- `_config.yml`: 블로그 설정 (제목, 테마 등)

파일을 수정한 후 GitHub에 푸시하면 자동으로 블로그에 반영됩니다!
